## 2. Що змінено / виправлено і чому

| Зміна | Причина |
|---|---|
| Усі посилання на `cdn.prod.website-files.com` (HTML + CSS `url()`) → локальні відносні шляхи | Після скасування Webflow його CDN перестане віддавати файли |
| Асет-теки перейменовано на нейтральні (`assets/cdn/`, `assets/js/`) | Щоб і в локальних шляхах не лишалося назв Webflow-доменів |
| jQuery локалізовано (`assets/js/jquery-3.5.1.min.js`) | webflow.js (анімації, мобільне меню) залежить від jQuery з CloudFront Webflow |
| Webflow-бейдж, виклики `editor-api.webflow.com` / `webflow.com` / badge-SVG — нейтралізовано | Прибрати зовнішні звернення до Webflow і сам бейдж |
| Структура `шлях/index.html`, внутрішні посилання **відносні** | URL 1:1 (SEO/зовнішні посилання) + працює на будь-якому базовому шляху |
| **12 сторінок `map/…` додано** окремим краулом | Маркери карти вели на них, але лінки були лише в JS — перший краул пропустив |
| **Назви «Історичних періодів»** на головній — усі **12** пунктів мали шаблонні залишки (`Keto`, `Low carb`, `Paleo`, `Clean Eating`…) → замінено на справжні (з футера) | Пункти з демо-шаблону Webflow |
| Порожній слот-заглушка (`Clean Eating`, `href="#"`) → підключено до `osnovni-statti/ukrayina-v-pershi-povoienni-roki/` | Єдиний період без посилання в сітці |
| Форма «Додати історію» (`href="#"`) → `mailto:avtomaydanvin@gmail.com` (тема «Додати історію — memory.org.ua»), на всіх сторінках | За вашим рішенням |
| Кастомна `404.html`; `CNAME` = `www.memory.org.ua` | 404 у стилі сайту; кастомний домен |
| Google Tag Manager (`GTM-WFQCCNH`), gtag | **Залишено як є** (за вашим проханням) |

## 3. Відомі обмеження (що інакше, ніж на Webflow)

- **Карта `/some-map`** — Leaflet (custom embed). **Повністю self-hosted з 2026-07-23:** 1117 тайлів + Leaflet перенесено в репозиторій (`static/`), зовнішніх залежностей у карти більше немає.
- **`archive.memory.org.ua`** — окремий проєкт, не чіпали; посилання абсолютні.
- **Форма «Додати історію»** тепер відкриває поштовий клієнт (`mailto`), а не вебформу. За бажання згодом — Google Form / Formspree.
- **Дрібні пре-існуючі заглушки** з Webflow («Збір коштів на ЗСУ» у футері, дисклеймер-текст у `<a href="#">`) лишені як в оригіналі. Деталі: [`qa-report.md`](qa-report.md).
- Редактор/CMS Webflow більше недоступний — правки контенту тепер через HTML у репозиторії.
