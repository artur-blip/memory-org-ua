## Деталі

### 1. Нуль Webflow-залежностей
Греп по всьому `site/`: `(website-files\.com|webflow\.com|cloudfront\.net)` → **0 збігів**.
- Усі посилання на `cdn.prod.website-files.com` у HTML + `url()` у CSS переписано на локальні відносні шляхи.
- Асет-теки перейменовано на нейтральні (`assets/cdn/`, `assets/js/`).
- Webflow-бейдж і виклики `editor-api.webflow.com` / `webflow.com` / badge-SVG з CloudFront — нейтралізовано в JS.
- jQuery (від нього залежить webflow.js) локалізовано з CloudFront → `assets/js/jquery-3.5.1.min.js`.

### 2. Приховані сторінки — знайдено і додано
- 12 сторінок `map/…` лінкуються лише зсередини JS-даних Leaflet-карти (маркери), тому їх не було в першому краулі. Додано окремим проходом. Детектор root-relative посилань підтвердив, що інших прихованих сторінок немає (0).

### 3. Структура URL 1:1
Кожна сторінка — `шлях/index.html`, старі URL відкриваються за тим самим шляхом (SEO/зовнішні посилання не ламаються). Внутрішня навігація — **відносна** (працює на `*.github.io/repo/`, кастомному домені й локально).

### 4. Мережа (DevTools)
Лише власний домен + дозволені зовнішні (не-Webflow): Google Fonts (через webfont.js, protocol-relative → на HTTPS стає `https`) і Google Tag Manager/gtag (`GTM-WFQCCNH`, лишено за проханням). Leaflet і тайли карти вантажаться зі свого ж домену (`/static/`).
